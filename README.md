# GPaint

*A simple drawing program made in Libadwaita and GTK4*

# Description
Paint clone made to look and feel exactly like modern Gnome apps. You can, of course, load and save images, and edit them.

Featuring the following tools and features:
- rectangle and free selection, with option to invert colors and such
- import images in others
- create new canvases with transparent background, or a background of your choice
- open or save files in PNG, JPEG or BMP, or open images with other apps
- Brush, airbrush, calligraphic, pencil with options for everything
- intelligent fill with gradient filling also
- insert text in the canvas with options
- resize the canvas directly with pixels or centimeters, or directly from the canvas with the points

# Build and installation
This app was made in Gnome Builder, so download the repository and open it in Gnome Builder. Once done so, you should be able to hit "Play" and it works.

App is distributed under releases as an installable flatpak.
